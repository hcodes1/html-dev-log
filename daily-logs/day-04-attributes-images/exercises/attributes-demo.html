HTML Attributes

Attributes give us more control and detail about HTML elements. Here are some examples:

Anchor Tag () with href and target attributes:

The `href` attribute specifies the URL the link goes to.
The `target` attribute specifies where to open the linked document (e.g., `_blank` opens in a new tab).

Visit Google in a new tab

Input Tag () with type, id, and name attributes:

The `type` attribute defines the type of input (e.g., `text`, `email`, `password`).
The `id` attribute provides a unique identifier for the element, useful for CSS and JavaScript.
The `name` attribute is used when submitting form data.
